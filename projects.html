Projects & Code

Here are some of my key projects, simulation models, and toolkits developed as part of my PhD research. Each project highlights objectives, methodology, and links to the code or visual results.

Quarter Car Suspension Model

A modular Python/Simulink model to simulate vertical dynamics of a quarter car, evaluating ride comfort and suspension performance under various road profiles.

View Code on GitHub

Bicycle Model (Pitch + Vertical Dynamics)

Simulation of a 2-DOF bicycle model including pitch and vertical motion, used to study ride comfort, safety metrics, and chassis dynamics for mid-size vehicles.

View Code on GitHub

Vehicle Control Toolkit (Python)

A comprehensive Python toolkit for implementing, simulating, and comparing control strategies: PID, LQR, MPC, Skyhook, Sliding Mode, Fuzzy Logic, and hybrid controllers.

View Toolkit on GitHub
